# AI Based Cash Flow Management System

## 📌 Description
This project is an AI-based web application that manages income and expenses while predicting future cash flow using historical financial data.

## 🚀 Features
- Income and expense tracking
- Cash flow prediction using historical data
- Real-time balance updates
- Interactive dashboard for financial insights

## 🛠 Tech Stack
- Python
- Django
- PostgreSQL / MySQL
- HTML, CSS

## ▶️ How to Run
1. Clone the repository
2. Install dependencies: pip install -r requirements.txt
3. Run the server: python manage.py runserver

## 📊 Use Case
Helps users manage finances efficiently and make better financial decisions using predictive analytics.
